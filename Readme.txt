Активация Clion 2017.2 — 2017.3 локально.

1.	Загружаем файл LocalServer.zip

2.	Распаковываем архив и выбираем файл с названием вашей операционной системы

3.	Даем права на запуск файлу и запускаем файл


4.	Выбираем License server и в поле License server address: указываем адрес http://127.0.0.1:1017/


5.	Нажимаем кнопку Activate.
